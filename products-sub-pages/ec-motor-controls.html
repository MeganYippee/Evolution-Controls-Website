EVO/™ECM-3PT (3 Point Floating Control Unit)
  • Increase/decrease control for EC Motors.
  • Adjustable Rate.
  • 0-10V feedback.
  • Status and Flow Index Lamps.
  • OEM and Contractor Versions.
  • Allows VAV controls to control motor instead of a damper.
DATASHEET


EVO/™ECM-4Spd-PRO (4 Speed Interface Unit)

Allows 24 Vac controls to select one of four adjustable flow indexes for an EC Motor.

  • Isolated Taps.
  • Fast or Gradual Speed Change.
  • Adjustment range is from Off to 100% of the programmed control range.
  • A fifth adjustment sets the flow index when no call is selected.
  • Pilot Pulse Enable/Disable.
  • PWM or 0-10 V output

EVO/™ECM-ACU-PRO (Automation Control Unit)

Allows industry standard 0-10V controllers to adjust and monitor the EC Motor.

  • Allows remote adjustment of output from 0 to 100% of programmed control range.
  • Fan flow can be factory set using manual adjust and heating can be tested without connecting a 1-10V supply.
  • Lamp indicators reference flow index.
  • During construction, heat can be used before automation is installed.
  • Air balance can be achieved before automation is installed.
  • Crystal controlled RPM accuracy.
